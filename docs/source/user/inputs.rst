Advanced Inputs
===============

This section describes the full configuration options available in SimDesign.
Inputs are provided through the ``inputs`` dictionary passed to
:func:`simdesign.rcmrf.generate`.

.. code-block:: python

    inputs = {
        "bcim": {...},
        "bnsm": {...}
    }
